# **Folhastech Open Hardware**

Acreditamos que o hardware deve ser um recurso valioso e acessível, capaz de inspirar o aprendizado e a inovação. Procuramos projetar soluções que possam ter uma vida útil prolongada e, sempre que possível, ser adaptadas ou reaproveitadas para novos propósitos.
